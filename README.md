# manuscript-from-figures-skill
This skill turns a figure set plus scattered source materials into a complete, evidence-preserving biomedical manuscript draft.

Use it when the user provides main figures, original manuscripts, rewritten drafts, supplementary files, PDFs, or experimental notes and asks for a full paper, a bilingual draft, or a deeper rewrite of Introduction, Results, Discussion, Methods, and figure legends.

For the full workflow and reusable checklists, read references/workflow.md at the start of a full manuscript build or whenever the user asks for a process document.

Core Principle
Write from the evidence ledger, not from memory or generic paper templates. Every figure panel, quantitative result, dose, instrument parameter, statistical method, abbreviation, and method detail must either appear in the manuscript or be explicitly marked as missing/uncertain.

Required Inputs To Inventory
At the start, identify and list:

Main figures and all panels: Fig. 1A, 1B, etc.
Existing manuscript drafts in all languages.
Supplementary materials, tables, raw result summaries, and methods notes.
Reference PDFs or target journal examples for structure, tone, and layout.
User-supplied corrections: dose units, model conditions, ethics numbers, instrument models, calibration ranges, statistics, group names, and figure order.
Workflow
Extract text and images from all Word/PDF/source files. Render documents or figures when layout matters.
Build a figure-panel evidence ledger: for each panel, record what it shows, the source evidence, key numbers, method dependencies, and required caption details.
Build the paper story line from the figure order: model and efficacy, distribution, omics screening, targeted validation, spatial localization, and mechanism.
Draft the manuscript structure before prose: title, abstract, introduction, results with figures after each result, discussion, materials and methods, figure legends, references, and supplementary notes.
Write Results panel-by-panel. Each result section must name the relevant figure panels and explain the experimental meaning, not just summarize the figure title.
Write a classic four-paragraph Introduction: field problem, candidate molecule and evidence gap, model/technology rationale, study aim and novelty.
Write a layered Discussion: main finding, model interpretation, exposure/distribution, mechanistic branches, spatial evidence, computational support, limitations, and future validation.
Write Materials and Methods as formal subsections. Do not collapse experiments into one paragraph.
Write figure legends with every panel, abbreviation definitions, group information, sample size/statistical method when available, and significance notation.
For bilingual work, write the English scientific version first, then produce a faithful Chinese version that preserves technical details, numbers, figure references, and logic.
Validate against the evidence ledger and render/check the final DOCX if a Word deliverable is produced.
Non-Negotiable Checks
No main figure panel is left unexplained in Results or figure legends.
Results mention panel labels in order, for example "Fig. 2A-G".
Every figure is placed after its corresponding result section when the requested layout requires it.
Discussion does not reduce months of experiments to a few generic sentences.
Methods include animals, ethics, grouping, dosing, modeling, sampling, instruments, chromatography/MS settings, omics/statistics, imaging, and software when available.
Units are normalized across the manuscript.
Any unknown detail is flagged as "to be confirmed" instead of invented.
Output Style
Prefer a complete draft over a high-level outline unless the user asks only for planning. Preserve the user's experimental details and requested terminology. When improving an existing manuscript, edit in place and report exactly which files were updated and how validation was performed.
